## Building a LearnDash Site

Prerequisites: A WordPress website, administrator access, a selected course structure, and a plan for lessons, quizzes, certificates, payments, and learner communication.

1.  **Install and Activate:** Add the learndash wordpress plugin to WordPress, activate the license, and confirm that core course settings match the site plan.
2.  **Create Course Content:** Build LearnDash courses with lessons, topics, LearnDash quiz settings, assignments, prerequisites, and completion rules.
3.  **Configure Access and Sales:** Connect learndash woocommerce or another checkout method, then set enrollment rules, pricing, coupons, and course visibility.
4.  **Publish and Test:** Preview LearnDash certificates, learner progress, group access, LearnDash login behavior, email messages, and mobile layouts before launch.

## Fixing Common Setup Problems

- Course pages not appearing? Refresh WordPress permalinks, review learndash documentation, and confirm that course visibility and enrollment settings are correct.
- Checkout not enrolling users? Recheck learndash woocommerce product links, payment status rules, user roles, and course access settings.
- Learners cannot sign in? Test the LearnDash login flow, password reset emails, caching rules, and any membership plugin restrictions.
- Certificates not generating? Confirm LearnDash certificates are assigned to the right course or quiz, then verify completion rules and PDF display settings.
